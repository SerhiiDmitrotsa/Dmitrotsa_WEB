Назва атрибута <form>

HTML атрибут <action> указывает обработчик, к которому обращаются данные формы при их отправке на сервер. В качестве обработчика может выступать CGI-программа или HTML-документ, который включает в себя серверные сценарии (например, Parser). После выполнения обработчиком действий по работе с данными формы он возвращает новый HTML-документ.

Синтаксис атрибута

<FORM ACTION="url"> ... </FORM>

Значение

Значение Описание
URL Определяет адрес, куда отправляется форма (по умолчанию действие установлено на текущую страницу). Допускаются как абсолютные, так и относительные URL адреса.

Браузеры

Internet Explorer Chrome Opera Safari Firefox Android Apple
3.0+ 1.0+ 4.0+ 1.0+ 1.0+ 1.0+ 1.0+

Пример:


<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Тег FORM, атрибут action</title>
    </head>
    <body>
    <form action="handler.php">
    <p>...</p>
    </form>
    </body>
</html>
